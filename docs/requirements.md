# 八卦おみくじアプリ 要件定義書

## 1. プロジェクト概要

### 成果目標
生年月日から算出した十干タイプ×ランダム八卦で「今日の自分の使い方」を儀式体験として1クリックで伝える、誰でも無料で使えるWebアプリ。結果画面から講座への動線を設置する。

### 成功指標（定量）
- 操作ステップ3以内（入力 → 念じる → 結果）
- 統合メッセージ80通り（十干10 × 八卦8、全組み合わせ網羅）
- ガチャ演出込みで10秒以内に結果表示
- 生年月日をlocalStorageに保存し、2回目以降は入力スキップ

### 成功指標（定性）
- 筮竹アニメーション・音で「引いた感」「儀式感」を感じられる
- 占いの知識ゼロでも直感的に自分の状態を内省できる
- 講座への動線として自然に機能する世界観

### 十干（10種）
| 干 | タイプ名 | 説明 |
|----|---------|------|
| 甲 | まっすぐ成長型 | 真っ直ぐ上に向かう大木のエネルギー |
| 乙 | 柔軟調整型 | 風にしなやかに揺れる草花のエネルギー |
| 丙 | 発信・太陽型 | 周囲を明るく照らす太陽のエネルギー |
| 丁 | 繊細表現型 | 静かに灯り続けるろうそくのエネルギー |
| 戊 | 安定・守り型 | どっしりと構える大地のエネルギー |
| 己 | 育成サポート型 | 万物を育む豊かな田畑のエネルギー |
| 庚 | 決断・突破型 | 鋭く断ち切る刀のエネルギー |
| 辛 | 洗練・美意識型 | 美しく磨かれた宝石のエネルギー |
| 壬 | 流れ・柔軟型 | 大海原を流れる水のエネルギー |
| 癸 | 感受・浸透型 | 大地にしみこむ雨のエネルギー |

### 八卦（8種）
| 卦 | 名前 | 今日の意味 |
|----|------|-----------|
| 乾 | 行動・リーダー | 前に出る、決断する日 |
| 坤 | 受容・育成 | 受け入れ、育てる日 |
| 震 | スタート・変化 | 動き出す、変化を起こす日 |
| 巽 | 広がり・柔軟 | 流れに乗る、広げる日 |
| 坎 | 慎重・内省 | 立ち止まる、深める日 |
| 離 | 表現・発信 | 輝く、伝える日 |
| 艮 | 停止・安定 | 整える、待つ日 |
| 兌 | 喜び・交流 | 楽しむ、つながる日 |

### 統合メッセージ構造
- 十干10種 × 八卦8種 = **80通り**
- 1メッセージ = 60〜80字
- 構成：「あなたの性質（十干）＋今日の流れ（八卦）＋具体的な行動ヒント」

**例：乙（草花タイプ）× 坎（慎重・内省）**
> あなたは柔らかく調整する力を持っています。今日は前に出るより、一度立ち止まり整えることで流れが良くなります。

## 6. 技術設計

### 十干計算ロジック
既存の命式ソフト（`/home/hinoe/四柱推命命式/am.html`）から以下を流用：

```javascript
// ユリウス日数計算
function calcJDN(y, m, d) {
  if (m <= 2) { y--; m += 12; }
  const A = Math.floor(y / 100);
  const B = 2 - A + Math.floor(A / 4);
  return Math.floor(365.25 * (y + 4716)) + Math.floor(30.6001 * (m + 1)) + d + B - 1524;
}

// 日柱計算 → 日干（十干）を取得
function calcNicchuu(year, month, day) {
  const jdn = calcJDN(year, month, day);
  const idx = ((jdn + 49) % 60 + 60) % 60;
  return KAN[idx % 10]; // 甲〜癸の1文字
}
```

時間入力不要。生年月日のみで日干が確定する。

### 筮竹アニメーション
- CSS animation + JavaScript制御
- 棒（細い縦線）を複数本表示 → シャッフルアニメーション → 収束
- 演出時間：3〜5秒

### 効果音
- Web Audio API（外部ファイル不要）またはMP3ファイル
- シャッフル音（カラカラ）
- 確定音（チン）
